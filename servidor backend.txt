para conectarnos en este caso usaremos un servidor backend(del lado del servidor):
primero pondremos en el terminal:  npm init  ,despues pondriamos:  npm install express --save  ,a continuacion: npm install mongoose --save
despues: npm install body-parser --save 
creamos un archivo para la entrada del servidor:app.js
